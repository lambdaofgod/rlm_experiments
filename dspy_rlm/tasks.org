#+title: DSPy SFT Data Collection - Tasks

* Overview

Collect SFT training data from a DSPy program (teacher LM), filter by
metric, and write JSONL in DSPy's ChatAdapter format -- ready for
finetuning with Unsloth/TRL. Based on the ~SFTDataCollector~ pattern
from [[file:dspy_sft_data_collection.org][dspy_sft_data_collection.org]]. The DSPy module type
(ChainOfThought, RLM, etc.) is configurable via YAML.

** TODO Parse RLM output and evaluate with LongBench-Pro metrics

   RLM output (SFT JSONL trajectory) contains SUBMIT calls with answers
   that may need parsing/normalization.  LongBench-Pro uses 6 different
   metrics depending on ~secondary_task~.  We need a script that:

   1. Parses RLM answers from the collected SFT data (extract the value
      passed to ~SUBMIT(answer=...)~ -- handle both literal values and
      variable references like ~SUBMIT(answer=answer)~)
   2. Joins predictions back to the dataset to get gold answers and
      ~secondary_task~
   3. Evaluates using the correct metric per question

*** Metric mapping (~secondary_task~ -> metric)

    | secondary_task | Metric            | Description                                          |
    |----------------+-------------------+------------------------------------------------------|
    | T1.1, T1.2     | NDCG              | Ranking quality; gold positions = relevance scores    |
    | T2.1, T2.2     | Pairwise_Accuracy | Fraction of item pairs in correct relative order      |
    | T3.1, T3.2     | Accuracy          | Exact match on first element only                    |
    | T4.1, T4.2     | Summary           | 0.5*SemSim + 0.5*ROUGE-L (needs embedding model)    |
    | T5.1, T5.2     | F1_Score          | Set-level F1 over answer elements                    |
    | T6.1           | SubEM             | Fraction of gold elements present in prediction      |
    | T6.2           | F1_Score          | Set-level F1                                         |
    | T6.3           | Pairwise_Accuracy | Pair ordering                                        |
    | T7.1-T7.3      | F1_Score          | Set-level F1                                         |
    | T8.1-T8.3      | SubEM             | Subset exact match                                   |
    | T9.1, T9.2     | F1_Score          | Set-level F1                                         |
    | T10.1, T10.2   | SubEM             | Subset exact match                                   |
    | T11.1, T11.2   | Accuracy          | Exact match on first element                         |
